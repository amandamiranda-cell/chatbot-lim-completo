# Chatbot LIM

Chatbot de navegação por áreas do LIM-HC, com FAQs e envio de dúvidas por e-mail.

## Funcionalidades

- Navegação: **Área → Assunto → Pergunta → Resposta**
- Respostas com **Markdown**, links e PDFs do portal
- Formulário de contato que gera **mailto:** para envio de dúvidas
- Fácil manutenção: edite o JSON `faq_chatbot_lim.json` para atualizar perguntas e respostas
